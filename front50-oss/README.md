## Configuring OSS store for front50

#### OSS: 
```yaml
  oss:
    enabled: true
    endpoint: oss-cn-hangzhou.aliyuncs.com
    bucket: 
    accessKeyId: 
    accessSecretKey: 
    versioning: true
    readOnlyMode: false 
```
